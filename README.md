## ✨ Sobre o Projeto

Site pessoal e portfólio profissional de **Andre Silva** — Desenvolvedor FullStack baseado no Japão, especializado em criar soluções digitais sob medida para freelancers e empreendedores.

SPA (Single Page Application) com design dark premium, acentos dourados e suporte completo a três idiomas.

## 🚀 Funcionalidades

- **🌍 Multilíngue** — Alternância instantânea entre PT-BR, EN e JP com persistência
- **🎨 Design Premium** — Tema dark sofisticado com acentos dourados e gradientes
- **📱 100% Responsivo** — Otimizado para mobile, tablet e desktop
- **⚡ Performance** — SPA em arquivo único, carregamento ultrarrápido
- **💬 WhatsApp Direto** — Formulário inteligente que gera mensagem formatada no WhatsApp
- **📊 Estimativas Automáticas** — Geração de estimativas baseadas no tipo de projeto
- **🔍 SEO Otimizado** — Meta tags, Open Graph e estrutura semântica
- **✨ Animações** — Fade-in on scroll, contadores animados, efeitos hover premium
